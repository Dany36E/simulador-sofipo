#  Simulador de Inversiones Multi-SOFIPO

Simulador interactivo para comparar rendimientos de inversiones en diferentes SOFIPOs mexicanas (Nu, DiDi, Stori, Klar, Ualá, Mercado Pago, Finsus).

##  Características

- Comparación de 7 SOFIPOs reguladas por CNBV
- Cálculo de rendimientos con interés compuesto y simple
- Visualización de proyecciones a 3, 6, 12 y 24 meses
- Análisis de diversificación de portafolio
- Estrategias de inversión (Conservadora, Balanceada, Agresiva)

##  SOFIPOs Incluidas

- **Nu México**: Hasta 15% en Cajita Turbo
- **DiDi**: 16% primeros \
- **Klar**: 15% con membresía Plus/Platino
- **Stori**: 13.5% con tarjeta de crédito
- **Mercado Pago**: 12.5%
- **Ualá**: 10-11%
- **Finsus**: 11%
